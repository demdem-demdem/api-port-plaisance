# API Port Plaisance

This was an Homework for the CEF, I created an API to create and modify catways and reservations

To get started with this project, follow these steps:

1. Install the dependencies with:
   ```bash
   npm install
   ```

2. Create a env folder then add the following environment variables to your .env file:
(note: You can create a .env.dev and .env.prod to change the variables :) )
   ```env
URL_MONGO=<Your MongoDB URL>
SECRET_KEY=<secret key>
   ```

3. Now you can use :
```bash
npm run start
```
To use the API!

4. And voila.
