# Vue d'ensemble de l'API Russell

Bienvenue dans la documentation officielle du port de plaisance. 
Cette API permet de gérer les catways (emplacements de parking pour bateaux), 
les réservations associées ainsi que les utilisateurs du système.
L'accès est protégé par authentification JWT.
