# Glossaire technique et métier

| Terme | Définition |
| :--- | :--- |
| **Catway** | Un petit ponton perpendiculaire au quai principal, permettant l'amarrage de deux bateaux. |
| **Jeton (Token) JWT** | "JSON Web Token". Un jeton sécurisé envoyé par le serveur pour identifier l'utilisateur. |
| **Bcrypt** | Algorithme utilisé pour hacher (sécuriser) les mots de passe. |
| **Middleware** | Fonction intermédiaire qui s'exécute avant la logique finale d'une route (ex: vérification de connexion). |
| **CRUD** | Create, Read, Update, Delete. Les quatre opérations de base de la gestion de données. |
| **Endpoint** | Point d'entrée d'une API (une URL spécifique). |
| **Mongoose** | Bibliothèque utilisée pour faire le lien entre Node.js et MongoDB. |
